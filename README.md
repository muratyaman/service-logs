# service-logs
Microservice to manage logs. It's using Node.js, Express, Mongoose and Mongodb.

## Requirements
* node
* npm
* express
* mongodb
* mongoose
* body-parser
* compression

## Installation
1. run: `npm install`
2. create database 'service-logs' in mongodb
3. create collection 'logs' in mongodb

## Usage
`npm start`

## Manual tests
Import and use postman collection file.
